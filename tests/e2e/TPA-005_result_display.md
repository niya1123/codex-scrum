# TPA-005 結果表示画面

- 担当: Dev-FE, QA
- 目的: 日見出し配下に3カード（朝→昼→夕）、3日で9カード。条件変更で再検索可能

## 前提
- `aria-label="result"` セクションに結果を描画

## 操作
1) Happy Pathで検索 → 結果表示
2) 3日入力時にカードが9件であることを確認
3) 行き先か日付を変更して再検索 → 結果が更新される

## 期待結果
- 日別の見出し配下に3カード（順序: morning → afternoon → evening）
- 3日入力で9カード
- 再検索後、結果が更新される（前回結果はクリア）
