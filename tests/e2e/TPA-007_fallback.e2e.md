# TPA-007 タイムアウト&フォールバック（ダミー）E2E雛形

テスト名: E2E-TPA-007-01 DELAY_MS=5000 で fallback を返し 3 秒以内描画

- 前提:
  - BE に内部遅延を与えるための設定がある（例: 環境変数 `DELAY_MS`）。
  - BE の内部タイムアウトは 2.5s、タイムアウト時は `source:"fallback"` を付与して 200 を返す。

- 操作:
  1) BE を `DELAY_MS=5000` で起動（もしくは同等の遅延を注入）。
  2) フロントでフォーム送信し、クリック時刻を記録。

- 期待結果:
  - 初回描画が 3.0s 以内に行われる。
  - 表示された JSON/結果には top-level `source:"fallback"` が含まれる。
  - 通常 UI と同一表示（任意で「プレビュー」バッジ）。
