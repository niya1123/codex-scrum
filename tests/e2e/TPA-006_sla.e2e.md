# TPA-006 SLA: 3秒以内に応答 E2E雛形

テスト名: E2E-TPA-006-01 クリックから初回描画まで ≤ 3.0s

- 前提:
  - 通常時もしくは人工遅延（≤2.5s）を付与した環境。

- 操作:
  1) フロントでフォーム送信（クリック時刻を記録）。
  2) 初回描画（結果ヘッダや1枚目のカード）が DOM に現れた時刻を記録。

- 期待結果:
  - `初回描画時刻 - クリック時刻 ≤ 3000ms`。
  - 3秒を超える場合でも、TPA-007 のフォールバック動作により 3秒以内に初期結果が表示。

- 補足:
  - 高分解能タイマ（`performance.now()` 等）で計測。
