# TPA-007 タイムアウト＆フォールバック（ダミー）

- 担当: Dev-BE, Dev-FE, QA
- 目的: BE内部タイムアウト=2.5s超過で決定的ダミーに切替、3秒SLAを満たす

## 前提
- 環境変数例: `DELAY_MS` と `PLAN_TIMEOUT_MS=2500` を導入（テスト時に上書き）

## 操作
1) `DELAY_MS=5000` で検索実行
2) クリック時刻から結果初回描画までの時間を計測

## 期待結果
- 3.0s以内に結果が描画される
- レスポンスに `source: "fallback"` が含まれる（200で返却）
- FEは通常UIで表示（任意で「プレビュー」バッジ）
