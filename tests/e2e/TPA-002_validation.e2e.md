# TPA-002 即時バリデーション（理由付き）E2E雛形

テスト名: E2E-TPA-002-01 空destinationでエラー（100ms以内表示・API呼び出し0回）

- 前提:
  - ルート画面でフォームが表示されている。

- 操作:
  1) destination を空のまま focus → blur。

- 期待結果:
  - 「行き先は必須です」のエラーが 100ms 以内にインライン表示。
  - ネットワーク呼び出し回数は 0。

テスト名: E2E-TPA-002-04 start_date > end_date でエラー（API呼び出し0回）

- 前提:

- 操作:
  1) start_date に `2025-10-10`、end_date に `2025-10-05` を入力。
  2) 送信ボタンをクリック（もしくは blur）。

- 期待結果:
  - 「終了日は開始日以降にしてください」が 100ms 以内に表示。
  - ネットワーク呼び出し回数は 0。
